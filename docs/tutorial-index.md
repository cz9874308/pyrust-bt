## ❓ 常见问题

### 安装问题

**Q: Rust 编译失败怎么办？**  
A: 检查 Rust 版本（需要 1.70+），确保网络连接正常，可能需要配置国内镜像源。

**Q: maturin 安装失败？**  
A: 确保 Python 版本 >= 3.8，使用 `pip install maturin` 安装，如果失败可以尝试使用 conda。

**Q: 找不到 engine_rust 模块？**  
A: 需要先编译 Rust 引擎，运行 `cd rust/engine_rust && maturin develop --release`。

### 运行问题

**Q: 数据文件找不到？**  
A: 检查 CSV 文件路径是否正确，确保文件格式符合要求（datetime,open,high,low,close,volume）。

**Q: 回测结果异常？**  
A: 检查策略逻辑是否正确，确认数据格式，查看错误日志。

**Q: 性能很慢？**  
A: 尝试增大 `batch_size` 参数，使用 DuckDB 存储数据，使用向量化指标计算。

### 概念理解

**Q: 什么是回测？**  
A: 回测就是用历史数据验证交易策略的有效性，就像"时间机器"回到过去测试策略。

**Q: Strategy 和 BacktestEngine 的区别？**  
A: Strategy 定义交易逻辑（做什么），BacktestEngine 执行回测（怎么做）。

**Q: 市价单和限价单的区别？**  
A: 市价单立即成交（按当前价格），限价单指定价格成交（可能不成交）。
